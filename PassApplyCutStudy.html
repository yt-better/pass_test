应用统计学 计算题整理

一、第五章 抽样分布

第1题

📌 题目

一个具有 n=64 个观察值的随机样本抽自均值等于 20，标准差等于 16 的总体。要求：

  1. 计算 x̄ 的抽样分布（重复抽样）的均值和标准差
  2. 描述 x̄ 的抽样分布的形状
  3. 计算 x̄ < 16 的概率
  4. 计算 x̄ > 23 的概率
  5. 计算 x̄ 落在 16~22 范围内的概率

✅ 答案

本题公式

标准正态分布：

Z = (x̄ - μ) / σ_x̄ ~ N(0,1)

1. 抽样分布的均值和标准差

均值：μ = 20

标准差：σ_x̄ = σ/√n = 16/√64 = 16/8 = 2

2. 抽样分布的形状

钟形曲线，X ~ N(20, 2²)

3. x̄ < 16 的概率

P(Z < (16-20)/2) = P(Z < -2) = 0.0228

4. x̄ > 23 的概率

P(Z > (23-20)/2) = P(Z > 1.5) = 1 - 0.9332 = 0.0668

5. x̄ 落在 16~22 范围内的概率

P(-2 < Z < 1) = 0.8413 - 0.0228 = 0.8185

二、第六章 参数估计

第4题

📌 题目

某学校有 4500 名学生，采用不重复抽样随机抽取 20%，调查每月上网次数，其分布数列如下表所示：

上网次数0~2次2~4次4~6次6~8次8~10次
学生人数比重/%82240255

要求：

  1. 以 95% 的可靠性估计平均每名学生每月上网次数的区间范围
  2. 以 95% 的可靠性确定每月上网次数在 4 次以上比重的区间范围

已知：n = 900，N = 4500

✅ 答案

本题公式

公式表达式
加权平均数x̄ = Σ(fᵢxᵢ) / Σfᵢ
加权方差s² = Σfᵢ(xᵢ - x̄)² / Σfᵢ
不重复抽样标准误差u_x = √[(1-n/N) · s²/n]
极限误差Δ = Z · u_x
置信区间x̄ - Δ < X < x̄ + Δ

1. 平均上网次数的区间估计

组中值：x = [1, 3, 5, 7, 9]，f = [8, 22, 40, 25, 5]

平均数：

x̄ = (1×8 + 3×22 + 5×40 + 7×25 + 9×5) / 100 = 4.94

方差：

s² = [(1-4.94)²×8 + (3-4.94)²×22 + (5-4.94)²×40 + (7-4.94)²×25 + (9-4.94)²×5] / 100 = 3.9564

s = √3.9564 = 1.989

标准误差（不重复抽样）：

u_x = √[(1-900/4500) × 3.9564/900] = √[0.8 × 0.004396] = 0.0593

95% 可靠性，Z = 1.96：

极限误差 Δ = 1.96 × 0.0593 = 0.1162

置信区间：

P(4.94 - 0.1162 < X < 4.94 + 0.1162) = 4.82 ~ 5.06

2. 4 次以上比重的区间估计

样本比重：

p = (40+25+5)/100 = 0.7

标准误差：

u_p = √[(1-900/4500) × 0.7×0.3/900] = 0.0136

极限误差：

Δ = 1.96 × 0.0136 = 0.0267

置信区间：

P(0.7 - 0.0267 < P < 0.7 + 0.0267) = 67.33% ~ 72.67%

第9题

📌 题目

某厂生产的一批袋装奶的抽检结果如表所示（重复抽样）。这种袋装奶的标准质量每袋为 150g。

每袋质量/g148~149149~150150~151151~152
数量/袋10205020

要求：以 99.73% 的可靠保证程度

  1. 确定每袋奶质量的极限误差
  2. 估计这批袋装奶平均每袋的质量范围，确定是否达到规定质量的要求
  3. 若以每袋质量 150g 为合格，推算这批奶的合格率

✅ 答案

1. 极限误差

组中值：x = [148.5, 149.5, 150.5, 151.5]，f = [10, 20, 50, 20]，n = 100

平均数：

x̄ = (148.5×10 + 149.5×20 + 150.5×50 + 151.5×20) / 100 = 150.3g

方差：

s² = [(148.5-150.3)²×10 + (149.5-150.3)²×20 + (150.5-150.3)²×50 + (151.5-150.3)²×20] / 100 = 0.76

s = √0.76 = 0.871

标准误差（重复抽样）：

u_x = √(0.76/100) = 0.0871

99.73% 可靠性，t = 3：

极限误差 Δ = 3 × 0.0871 = 0.2613

2. 平均质量范围

P(150.3 - 0.2613 < X < 150.3 + 0.2613) = 150.04g ~ 150.56g

结论：平均质量范围包含 150g，达到规定质量要求。

3. 合格率

合格（≥150g）：50 + 20 = 70 袋

p = 70/100 = 0.7

标准误差：

u_p = √(0.7×0.3/100) = 0.0458

99.73% 置信区间：

P(0.7 - 3×0.0458 < P < 0.7 + 3×0.0458) = 56.26% ~ 83.74%

三、第七章 假设检验

第1题

📌 题目

一种罐装饮料采用自动生产线生产，每罐的容量是 255ml，标准差为 5ml。为检索每罐容量是否符合要求，质检人员在某天生产的饮料中随机抽取了 40 罐进行检验，测得每罐平均容量为 255.8ml。

要求：取 α = 0.01，检验当天生产的饮料容量是否符合标准要求。

✅ 答案

本题公式

总体方差已知，总体均值检验统计量：

Z = (x̄ - μ₀) × √n / σ

假设设定

扩展：H₁ 一般都是不含等于号的

H₀: μ = 255（符合标准）
H₁: μ ≠ 255（不符合标准）
双侧检验

计算检验统计量

Z = (255.8 - 255) × √40 / 5 = 0.8 × 6.3246 / 5 = 1.012

确定临界值

α = 0.01，双侧检验，α/2 = 0.005

临界值：±Z₀.₀₀₅ = ±2.576

决策

|Z| = 1.012 < 2.576

不拒绝 H₀

结论

在 α = 0.01 的显著性水平下，当天生产的饮料平均容量符合标准要求。

四、常用公式汇总

抽样分布

情形公式
重复抽样均值标准误σ_x̄ = σ/√n
不重复抽样均值标准误σ_x̄ = √[(1-n/N) · σ²/n]
标准正态变换Z = (x̄ - μ) / σ_x̄ ~ N(0,1)

参数估计

情形公式
均值置信区间x̄ ± Z · u_x
比例置信区间p ± Z · u_p
重复抽样比例标准误u_p = √[p(1-p)/n]
不重复抽样比例标准误u_p = √[(1-n/N) · p(1-p)/n]

假设检验

情形公式
总体方差已知，均值检验Z = (x̄ - μ₀)√n / σ
总体方差未知，均值检验t = (x̄ - μ₀)√n / s
比例检验Z = (p - p₀) / √[p₀(1-p₀)/n]

常用 Z 值

置信水平Z值说明
90%1.645单侧常用
95%1.96最常用
99%2.576α=0.01双侧
99.73%3近似3σ原则
