# 应用统计学 计算题

## 第五章

一个具体$n=64$个观察值的随机样本抽自均值等于$20$,标准差等于16的总体。要求
1. 计算$\bar{x}$的抽样分布(重复抽样)的均值和标准差
2. 描述$\bar{x}$的抽样分布的形状
3. 计算$\bar{x} \lt 16$的概率
4. 计算$\bar{x} \gt 23$的概率
5. 计算$\bar{x}$落在16~22范围内的概率


### 本题公式

1. 标准正太分布 $$Z = \frac{\bar{x} - \mu}{\sigma} \sim N(0,1)$$


### 解答

1. 均值 $$\mu = 20$$

标准差 $$\sigma = \sqrt{\frac{\sigma^2}{n}} = \sqrt{\frac{256}{64}} = 2$$

2. 形状钟形曲线 $$X \sim N(20,2^2)$$

3. $\bar{x} \lt 16$的概率$$P=({\frac{\bar{x} - \mu}{\sigma} \lt \frac{16 - \mu}{\sigma}}) = P(\frac{\bar{x} - 20}{2} \lt -2) = 0.0228$$

4. $\bar{x} \gt 23$的概率$$P=({\frac{\bar{x} - \mu}{\sigma} \lt \frac{23 - \mu}{\sigma}}) = P(\frac{\bar{x} - 20}{2} \gt 1.5) = 0.0668$$

5. $\bar{x}在 16 \sim 22$的概率为$$P =({\frac{16 - \mu}{\sigma} \lt \frac{\bar{x} - \mu}{\sigma} \lt \frac{22 - \mu}{\sigma}}) = P(-2 \lt \frac{\bar{x} - 20}{2} \lt 1) = 0.8185$$


## 第六章

### 4题

某学校有4500名学生，采用不重复抽样随机抽取$20\%$,调查每月上网次数，其分布数列如下表所示

|上网次数|0~2次|2~4次|4~6次|6~8次|8~10次|
|:---|:---|:---|:----|:-----|:----|
|学生人数比重/%|8|22|40|25|5|

### 要求

1. 以$95\%$的可靠性估计平均每名学生每月上网次数的区间范围 
2. 以$95\%$的可靠性确定每月上网次数在4次以上比重的区间范围

### 本题公式 n = 900 N = 4500

1. 加权 $$\bar{x} = \frac{\sum_{i=1}^{n} f_i x_i}{\sum_{i=1}^{n} f_i}$$

2. 非加权 $$\bar{x} = \frac{\sum_{i=1}^{n} x_i}{n}$$

3. 加权方差 $$\sigma ^2 = \frac{\sum_{i=1}^{n} f_i (x_i - \bar{x})^2}{\sum_{i=1}^{n} f_i}$$

4. 非加权 $$\bar{x} = \frac{\sum_{i=1}^{n} (x_i -\bar{x})^2}{n}$$

### 解答   无标准差用$s$

1. 平均数 $$\bar{x} =  \frac{8*1+3*22+5*40+7*25+9*5}{100} = 4.94$$

方差 $$ s^2 = \frac{(1-4.94)^2*8+(3-4.94)^2*22+(5-4.94)^2*40+(7-4.94)^2*25+(9-4.94)^2*5}{100}=3.9564$$
推出$s$:$$s=\sqrt{3.9564} = 1.989$$
$$u_x = \sqrt{(1-\frac{n}{N})*\frac{s^2}{n}} = 0.0593$$

极限误差 $$\Delta = 2 * u_x = 0.1186$$

质量范围 $$P=(\bar{x} - \Delta \lt \bar{X} \lt \bar{x} + \Delta)$$推出范围$$4.82 \sim 5.06$$

2. 四次以上的概率为 $$p = \frac{40+25+5}{100} = 0.7 $$
$$u_p = \sqrt{(1-\frac{n}{N})*\frac{p*(1-p)}{n}} =0.0136 $$

极限概率误差$$\Delta = 2*u_p = 0.0272$$

区间范围为$$P=(\bar{x} - \Delta \lt P \lt \bar{x} + \Delta)$$  范围：67.27%~72.73%

### 9题

某厂生产的一批袋装奶的抽检结果如表所示(重复抽样)。这种袋装奶的标准质量每袋为$150g$。

要求:以$99.73\%$的可靠保证程度

1. 确定每袋奶质量的极限误差
2. 估计这批袋装奶平均每袋的质量范围，确定是否达到规定质量的要求
3. 若以每袋质量150g为合格，推算这批奶的合格率

|每袋质量/g|数量/袋|
|:----|:----|
|148~149|10|
|149~150|20|
|150~151|50|
|151~152|20|
|合计|100|

### 本题公式

1. 加权 $$\bar{x} = \frac{\sum_{i=1}^{n} f_i x_i}{\sum_{i=1}^{n} f_i}$$

2. 非加权 $$\bar{x} = \frac{\sum_{i=1}^{n} x_i}{n}$$

3. 加权方差 $$\sigma ^2 = \frac{\sum_{i=1}^{n} f_i (x_i - \bar{x})^2}{\sum_{i=1}^{n} f_i}$$

4. 非加权 $$\bar{x} = \frac{\sum_{i=1}^{n} (x_i -\bar{x})^2}{n}$$


### 解答

1. 均值$\mu = 150$

平均数$\bar{x}$ 为 $$\bar{x} = \frac{148.5*10+149.5*20+150.5*50+151.5*20}{100}=150.3g$$

方差$s^2$ $$s^2=\frac{(148.5-150.3)^2*10+(149.5-150.3)^2*20+(150.5-150.3)^2*50+(151.5-150.3)^2*20}{100} = 0.76$$

$$s = \sqrt{s^2} = 0.871$$
$$u_x = \sqrt{\frac{s^2}{n}} = 0.0871$$

$99.73\%$ 此时t=3

极限误差为$$\Delta = tu_x = 0.2613$$

2. 平均质量范围是$$P(\bar{x}-\Delta \lt \bar{X} \lt \bar{x} + \Delta)$$ 范围:150.04~150.56

3. 大于等于150g质量为合格 $$p=\frac{50+20}{100}=0.7$$

$$u_p = \sqrt{\frac{p*(1-p)}{n}}=0.0458$$
$P({p-u_p \lt P \lt p+u_p})$ 范围:0.5653~0.8374


## 第七章 1题

一种罐装饮料采用自动生产线生产，每罐的容量是$255ml$,标准差为$5ml$。为检索每罐容量是否符合要求，质检人员在某天生产的饮料中随机抽取了40罐进行检验，测得每罐平均容量为$255.8ml$

要求:取 $\alpha = 0.01$，检验当天生产的饮料容量是否符合标准要求

### 扩展$H_1$一般都是不含等于号的

### 本题公式

总体方差已知，总体均值检验使用的统计检验计量公式
$$Z=\frac{(\bar{x}-\mu_0)*\sqrt{n}}{\sigma}$$

检验当天生产的饮料容量是否符合标准要求 得出双侧检验


### 解答

根据题意设:$$H_0 = 255,H_1 \neq 255$$

40下平均容量为$\bar{x}$=255.8ml
$$Z = \frac{(255.8-255)*\sqrt{40}}{5} = 1.012$$
已知$\alpha = 0.01$，那么$\frac{\alpha}{2}=0.005$
范围 =-2.576~2.576，然而$$Z \subseteq 这个区间 $$

不拒绝$H_0$假设

在$\alpha = 0.01$的显著性水平下，当天生产的饮料平均容量符合标准要求
